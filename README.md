# NLP Text Classification Project

This project demonstrates Natural Language Processing techniques for text classification using Python and machine learning.

## Project Overview

The project consists of two main tasks:

### Part A: IMDb Movie Review Sentiment Analysis
- Built a model to classify movie reviews as **positive or negative**
- Applied text preprocessing including tokenization, stopword removal, and stemming
- Used **TF-IDF vectorization**
- Implemented multiple models:
  - Logistic Regression
  - Naive Bayes
  - Support Vector Machine

Best Model: **Logistic Regression (~90% accuracy)**

---

### Part B: News Article Classification
- Classified news articles into multiple categories such as politics, sports, travel, etc.
- Performed text cleaning and preprocessing
- Applied **TF-IDF feature extraction**
- Trained multiple classification models

Best Model: **Logistic Regression (~65% accuracy)**
